# Personal Task Manager

A multi-user task management application built with FastAPI, SQLite, and Streamlit.

## Features

* User Registration
* User Login & Logout
* Create, View, Update, and Delete Tasks
* Mark Tasks as Done
* Filter Tasks by Status and Priority
* Task Summary Dashboard

## Tech Stack

* FastAPI
* SQLite
* Streamlit
* Requests
* Passlib (bcrypt)
* Python-dotenv

## Project Structure

* `backend/` – FastAPI backend and database logic
* `frontend/` – Streamlit frontend
* `designs.md` – Project planning and design
* `requirements.txt` – Project dependencies
